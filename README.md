# 🌍 Визуализация землетрясений: карта мира с данными GeoJSON

Интерактивная карта землетрясений, построенная на основе данных в формате GeoJSON. Проект использует Python и библиотеку Plotly для визуализации сейсмической активности.

## 🔎 Что делает проект

Программа:
- считывает данные о землетрясениях из файла `GeoJSON`;
- преобразует данные в читаемый формат;
- извлекает ключевые параметры: магнитуду, координаты и названия событий;
- строит интерактивную карту мира с маркерами землетрясений разного размера и цвета.

## 🛠️ Технологии и библиотеки

- **Python** — язык программирования;
- **Pathlib** — работа с файловой системой;
- **JSON** — парсинг данных;
- **Plotly Express** — создание интерактивной карты (`scatter_geo`).

## 📦 Структура проекта

* `data/d_30_day_m_1.geojson` — 🗺️ Исходные данные: землетрясения за 30 дней (GeoJSON).
* `data/readable_eq_data.geojson` — 🔎 Отформатированные данные для ручного просмотра.
* `eq_explore_data_0.py` — 🔎 Шаг 1: загрузка данных и первичный разбор.
* `eq_explore_data_1.py` — 🔎 Шаг 2: анализ структуры GeoJSON.
* `eq_explore_data_2.py` — 🔎 Шаг 3: извлечение координат и магнитуд.
* `eq_world_map_0.py` — 🌍 Шаг 4: базовая карта мира.
* `eq_world_map_1.py` — 🌍 Шаг 5: добавление маркеров землетрясений.
* `eq_world_map_2.py` — 🌍 Шаг 6: настройка размера маркеров (по магнитуде).
* `eq_world_map_3.py` — 🌍 Шаг 7: настройка цвета и подсказок.
* `eq_world_map_4.py` — 🌍 Шаг 8: финальная настройка стиля карты.
* `eq_world_map_5.py` — ✅ **Основной скрипт**: готовая интерактивная визуализация.
* `README.md` — 📝 Эта документация.

## 💾 Данные

Исходные данные взяты из общедоступного источника в формате **GeoJSON**. Файл `d_30_day_m_1.geojson` содержит информацию о землетрясениях за последние 30 дней с магнитудой ≥ 1.

Извлекаемые параметры:
- **Магнитуда** (`mag`) — размер и цвет маркера на карте;
- **Координаты** (`lat`, `lon`) — положение маркера;
- **Название события** (`title`) — отображается при наведении курсора.

## 🚀 Как запустить проект

### 🔧 Требования

Убедитесь, что у вас установлены:
- Python 3.8+;
- библиотеки: `plotly`, `pandas`

Установить зависимости можно командой:
    ```bash  
    pip install plotly

## 🚀 Запуск

1. Клонируйте репозиторий:
    ```bash
    git clone https://github.com/dshebeko20/new_recent_earthquakes.git
2. Перейдите в папку проекта:
    ```bash
    cd new_recent_earthquakes
3. Поместите файл d_30_day_m_1.geojson в папку eq_data/
4. Запустите скрипт:
    ```bash
   python eq_world_map_5.py
5. В браузере откроется интерактивная карта с данными о землетрясениях.

## 🎨 Настройка визуализации

Вы можете изменить:
-  color_continuous_scale — цветовую схему (например, 'Viridis', 'Plasma', 'Hot');
-  projection — проекцию карты ('natural earth', 'robinson', 'orthographic');
-  параметры маркеров (размер, прозрачность) через дополнительные аргументы px.scatter_geo.
